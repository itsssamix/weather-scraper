
Este projeto é uma automação desenvolvida em Python que coleta a temperatura atual de São Paulo através do site **weather.com**, registra o histórico em um arquivo Excel e oferece uma interface gráfica simples.

---

Tecnologias Utilizadas:

- Python 3
- Selenium WebDriver
- OpenPyXL
- Tkinter
- Logging
- ChromeDriver


Objetivo do Projeto:

- Coleta e processamento de informações externas
- Registro de histórico estruturado
- Interface para usuários não técnicos
- Organização modular do código

Este tipo de automação pode ser adaptado para monitorar:
- Sistemas internos
- Serviços externos
- Status de APIs
- Indicadores que afetam a operação da empresa
